JavaScript: Things to Remember

Common JavaScript Built-In Functions

  • toString() - Returns the string representation of a value
  • charAt() - Returns the character at the specified index
  • concat() - Combines the text of two strings
  • length() - Returns the length of a string
  • forEach() - Calls a function for each element in an array
  • splice() - Adds or removes elements from an array
  • substring() - Returns the characters in a string between two indexes in the string
  • indexOf() - Returns the first index of an element within an array equal to the specified value, or -1 if none is found
  • slice() - Extracts a section of an array and returns a new array
  • sort() - Sorts the elements of an array
Common Math Methods

  • Math.abs() - Returns the absolute value of a number
  • Math.ceil() - Returns the smallest integer greater than or equal to a number
  • Math.floor() - Returns the largest integer less than or equal to a number
  • Math.max() - Returns the largest of zero or more numbers
  • Math.min() - Returns the smallest of zero or more numbers
  • Math.pow() - Returns base to the exponent power, that is, base exponent
  • Math.random() - Returns a pseudo-random number between 0 and 1
  • Math.round() - Returns the value of a number rounded to the nearest integer

Accessing Elements in Nested Objects

Let's practice iterating over an array of objects: 


  var warriors = [{name: "Stephen Curry", position: "point guard"}, {name: "Klay Thompson", position: "shooting guard"}, {name: "Harrison Barnes", position: "small forward"}, {name: "Draymond Green", position: "shooting guard"}, {name: "Andrew Bogut", position: "center"}]
  for (var x in warriors){
  console.log(x)
  }
Based on our experience with Ruby, we might expect this to log information about each player. However, in JavaScript the variable "x" in warriors refers to the INDEX of each element and thus only logs the numbers 0-4. Let's modify this code a bit: 

  for (var x in warriors){
  console.log(warriors[x])
  }
This logs information (name and position) for each player. What if we just wanted to print their names out? 

  for (var x in warriors){
  console.log(warriors[x]["name"])
  }
This will print out the name of each player. Note the quotation marks around "name!"
